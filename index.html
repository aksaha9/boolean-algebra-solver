Boolean Algebra Software - developed by - Ashish Kumar Saha

Thank you for visiting this page to try out this software.

This Boolean Algebra software was created long back in 1991 by me - Ashish Kumar Saha - a student of 1992 pass out batch of Birla Vidya Mandir, Nainital, India using Pascal on MS-DOS. It takes any Boolean equation and is able to output the Truth table and minterm canonical forms. As my program is a 16-bit legacy DOS application, I have used Emscripten-compiled DOSBox to wrap it up and containerized it to then run in a web page.
